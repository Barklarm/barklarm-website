### Install

#### Using Package Manager

<Tabs>
  <TabItem value="osx" label="OSX" default>
    ```bash
        # Using brew
        brew install --cask barklarm/barklarm/barklarm
    ```
  </TabItem>
  <TabItem value="windows" label="Windows">
    ```bash
        # Using chocolatey
        choco install barklarm
    ```
  </TabItem>
</Tabs>
